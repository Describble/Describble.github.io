Describir Image Bundles

I made these image bundles using Videoscribe image library.

10k+ images bundle(Dropbox)

Be noticed, 'videoscribe_bundle_v1.zip' file is only for Describir1.2.0 and later.

Follow the steps bellow to import it.

  • Click the link to download the file 'videoscribe_bundle_v1.zip'.
  • Open Files app, move the zip file to the location 'On My iPhone/Describir/Describir'.
  • Uncompress the zip file.
  • Open Describir and enjoy.
