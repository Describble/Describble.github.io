How to install?

  • 1.Download the ZIP file on your device;
  • 2.Open the default Files application and move the ZIP file to the "On my iPhone/Flixcribir/Describir/Image Bundles" directory;
  • 3.Uncompress the ZIP file and the bunlde file will be in the same directory;
  • 4.Make sure the bundle file path is "On my iPhone/Flixcribir/Describir/Image Bundles/xxxxx.bundle";
  • 5.Open Flixcribir, the image bundle should be installed.
