Making Image Bundles

Need more images in Flixcribir? Now you can make your own image bundle.

▪️Step 1: Prepare your images

Each image for your bundle will require two files, a PNG file as the preview image and an SVG file as the image you want to use for editing, and both files must have the same name and be in the same directory，it will also work if you don't provide a PNG file, but you won't see the preview image.

Currently, only SVG files are supported.

▪️Step 3: Create image bundle

Open Flixcribir, click Create Image Bundle, select your image directory, and let Flixcribir do the rest.

▪️Done!

If your bundle is created correctly, it will load automatically,

and you can also find your image bundle file in the "On my iPhone/Flixcribir/Describir/Image Bundles" directory in the Files app.
